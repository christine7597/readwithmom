# ReadWithMom - Voice Cloning Audiobook

텍스트를 음성으로 변환하여 읽어주는 오디오북 웹 애플리케이션입니다.

## 기능

- ✅ 텍스트를 음성으로 변환 (Web Speech API)
- ✅ 다양한 언어 지원 (한국어, 영어, 일본어, 중국어, 스페인어, 프랑스어, 독일어)
- ✅ 읽기 속도 조절 (0.5x ~ 2.0x)
- ✅ 음성 선택 기능
- ✅ 일시정지/재개/중지 기능
- ✅ 예시 텍스트 제공

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 읽고 싶은 텍스트를 입력란에 입력
3. 언어와 음성을 선택
4. 읽기 속도 조절
5. "읽기 시작" 버튼 클릭

## 기술 스택

- HTML/CSS/JavaScript
- Web Speech API (SpeechSynthesis)

## 지원 브라우저

- Chrome/Edge (권장)
- Safari
- Firefox
